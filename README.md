# Next.js Steam Clone

Steam clone built with Next.js, uses auth and firebase rest api for login and app pricing, apps are managed via serverless jsons.

### Features
  - Login, Register
  - App Search
  - Dynamic App Page
  - Add Wish list
  - Add Cart

https://steam-klon.vercel.app/
